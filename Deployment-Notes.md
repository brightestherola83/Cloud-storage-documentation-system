# Deployment Notes

## Project: Cloud Storage & Documentation System

### 1. Created the S3 Bucket

I logged into the AWS Management Console and opened Amazon S3. I created a new S3 bucket to serve as cloud storage for school lecture materials.

### 2. Created Folders

Inside the S3 bucket, I created four folders to organize the school materials:

* Lecture-Notes
* Assignments
* Past-Questions
* Course-Materials

### 3. Uploaded Files

I uploaded sample educational materials into the appropriate folders. This helped demonstrate how school lecture materials can be stored and organized in Amazon S3.

### 4. Enabled Versioning

I enabled S3 Bucket Versioning to keep previous versions of files when they are updated or replaced. This provides an additional way to protect stored materials.

### 5. Created the GitHub Repository

I created a GitHub repository called **cloud-storage-documentation-system** to document the project and keep the project evidence in one place.

### 6. Uploaded Screenshots

I uploaded screenshots showing the S3 bucket, folder organization, uploaded files, and Versioning configuration into the GitHub repository.

### 7. Project Result

The project was successfully completed. Amazon S3 was used to provide cloud storage and organize school lecture materials, while GitHub was used to document the implementation and store screenshots and deployment notes.
